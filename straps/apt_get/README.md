# apt_get

| Attribute     | Value                                     |
|--------------:|-------------------------------------------|
| Namespace     | apt_get                          |
| Emoji         | An emoji that best describes your strap   |
| Description   | A description of your strap               |
| Dependencies  | ex: Git                                   |
| Compatability | ex: Unix, Mac OS                          |

## Configuration

```yml
apt_get:
  
  install:
    - { name: ruby }
```
